# ทะเลอลาสก้าของหนู (My Alaska Ocean) — Design Spec

วันที่: 2026-06-12
สถานะ: อนุมัติแนวคิดแล้ว รอเขียนแผนพัฒนา

## เป้าหมาย

เว็บเกม HTML5 เพื่อพัฒนาการเด็กอายุ 1-5 ปี ธีมสัตว์ทะเลอลาสก้า สองภาษาไทย/อังกฤษ
เล่นได้บน PC (เมาส์) และ iPad (สัมผัส) เผยแพร่ผ่าน GitHub Pages
ออกแบบให้เพิ่มมินิเกมใหม่ได้เรื่อย ๆ เมื่อเด็กโตขึ้น (รองรับถึง ~10 ปีในอนาคต)

## ขอบเขตเวอร์ชัน 1

### มินิเกม 4 เกม (ไล่ตามวัย)

1. **แตะทะเล (Tap the Sea)** — วัย 1-2 ปี ทักษะเหตุ-ผล
   - ฉากทะเลเต็มจอ แตะที่ว่างแล้วสัตว์โผล่มาพร้อมอนิเมชันและเสียง
   - แตะตัวสัตว์ → ขยับตัว ส่งเสียงร้อง และพูดชื่อสองภาษา ("วาฬเพชฌฆาต! Orca!")
   - ไม่มีถูก/ผิด ไม่มีคะแนน

2. **จับคู่เงา (Shadow Match)** — วัย 2-4 ปี ทักษะกล้ามเนื้อมัดเล็ก + การจำแนกรูปทรง
   - ลากสัตว์ 3-4 ตัวไปวางบนเงาที่ตรงกัน
   - วางถูก: เสียงเชียร์ + เอฟเฟกต์ฉลอง / วางผิด: ตัวสัตว์เด้งกลับเงียบ ๆ ไม่มีเสียงตำหนิ
   - ครบทุกตัว → ฉลองใหญ่ แล้วสุ่มชุดใหม่

3. **นับสัตว์ทะเล (Count & Tap)** — วัย 3-5 ปี ทักษะตัวเลข
   - สัตว์ว่ายอยู่ N ตัว ให้แตะตัวเลขที่ถูกจาก 3 ตัวเลือก
   - เริ่มช่วง 1-3 เพิ่มความยากอัตโนมัติถึง 10 เมื่อตอบถูกต่อเนื่อง (ลดลงเมื่อผิดบ่อย)
   - อ่านโจทย์ด้วยเสียงพูด ("มีปลาแซลมอนกี่ตัว?")

4. **เสียงเรียกใคร (Listen & Find)** — วัย 3-5 ปี ทักษะคำศัพท์สองภาษา
   - เสียงพูดชื่อสัตว์ (ภาษาตามโหมดที่เลือก) ให้แตะตัวที่ถูกจาก 3-4 ตัว
   - ตอบถูก: ฉลอง + พูดชื่อซ้ำทั้งสองภาษา / ตอบผิด: ตัวที่แตะส่ายหัวเบา ๆ แล้วให้ลองใหม่

### หน้าเมนูหลัก (Hub)

- ชื่อเกม + ปุ่มสลับภาษา ไทย/EN + ปุ่มเปิด-ปิดเสียง
- ปุ่มเกม 4 ปุ่มขนาดใหญ่ มีภาพสัตว์ขยับได้บนปุ่ม + ป้ายชื่อเกมสองภาษา + ช่วงวัยแนะนำ
- Mockup ที่อนุมัติแล้ว: `.superpowers/brainstorm/1605-1781235708/content/hub-mockup.html`

### สัตว์ 8 ตัวแรก

วาฬเพชฌฆาต (Orca), วาฬหลังค่อม (Humpback Whale), แมวน้ำ (Harbor Seal),
นากทะเล (Sea Otter), สิงโตทะเล (Sea Lion), นกพัฟฟิน (Puffin),
ปลาแซลมอน (Salmon), ปูอลาสก้า (Alaska King Crab)

## สไตล์ภาพ (ตัดสินใจแล้ว: แบบ D "ผสม")

- SVG วาดเองทั้งหมด ไม่ใช้ภาพสำเร็จรูป
- สัดส่วน/โครงอิงสัตว์จริง (แบบตัวอย่าง C) + สีเรียบ เงา 1-2 ชั้น ตาโตเด่นอ่านอารมณ์ง่าย (แบบตัวอย่าง A)
- ตัวอย่างที่อนุมัติ: `.superpowers/brainstorm/1605-1781235708/content/art-style-a-vs-c.html`
- ทุกตัวมีอนิเมชัน: ว่ายน้ำ (ลอยขึ้นลง/ครีบขยับ), กะพริบตา, ตอบสนองตอนแตะ (เด้ง/หมุนตัว)
- ฉากหลัง: ไล่สีน้ำทะเล แสงส่องจากผิวน้ำ ฟองอากาศ สาหร่าย พื้นทราย

## หลักออกแบบเพื่อเด็กเล็ก (บังคับใช้ทุกเกม)

- พื้นที่แตะขั้นต่ำ ~80px ทุกองค์ประกอบที่กดได้
- ไม่มีจับเวลา ไม่มีสถานะ "แพ้" ไม่มีเสียง/ภาพเชิงลบเมื่อตอบผิด
- ไม่มีตัวหนังสือเป็นเงื่อนไขการเล่น (เด็กยังอ่านไม่ออก) — ทุกคำสั่งมีเสียงพูด
- ไม่มีโฆษณา ไม่มีลิงก์ออกนอกเกม ไม่มี in-app purchase
- ปุ่มกลับเมนูเล็กไว้มุมบน (กันเด็กกดโดนบ่อย ๆ)

## สถาปัตยกรรม

- **Vanilla JS (ES Modules) + SVG ใน DOM** — ไม่มี framework ไม่มี build step ไม่มี dependency
- โครงไฟล์:
  ```
  index.html
  css/main.css
  js/main.js              — bootstrap + scene manager (hub ↔ game)
  js/core/audio.js        — Web Speech API (TTS ไทย/อังกฤษ) + Web Audio เอฟเฟกต์สังเคราะห์
  js/core/i18n.js         — ข้อความสองภาษา + ค่าภาษาปัจจุบัน
  js/core/animals.js      — ทะเบียนสัตว์: id, ชื่อ TH/EN, ฟังก์ชันสร้าง SVG, เสียงเอกลักษณ์ประจำตัว (สังเคราะห์จาก Web Audio ไม่ใช่เสียงร้องจริง)
  js/core/settings.js     — localStorage (ภาษา, เสียงเปิด/ปิด)
  js/games/tap-sea.js     — เกม 1
  js/games/shadow-match.js— เกม 2
  js/games/count-tap.js   — เกม 3
  js/games/listen-find.js — เกม 4
  ```
- ทุกเกม implement interface เดียวกัน: `{ init(container, ctx), destroy() }` — เพิ่มเกมที่ 5+ ได้โดยไม่แตะของเดิม
- สัตว์เป็นฟังก์ชัน JS คืน SVG element (ไม่ใช่ไฟล์ .svg แยก) เพื่อให้ปรับสี/ขนาด/อนิเมชันแต่ละส่วน (ครีบ ตา ปาก) ได้จากโค้ด

## Input และ Responsive

- ใช้ Pointer Events อย่างเดียว (ครอบคลุมเมาส์+สัมผัส) — ห้ามผูก mouse/touch events แยก
- แนวนอน (landscape) เป็นหลัก แสดงคำแนะนำหมุนจอเมื่อถือแนวตั้ง
- กันซูม/เด้งหน้าจอบน iPad: `touch-action: none` ในพื้นที่เกม + viewport meta + `user-scalable=no`
- รองรับ Add to Home Screen (meta apple-mobile-web-app)

## เสียง

- TTS: `speechSynthesis` เลือก voice `th-TH` / `en-US` ตามโหมดภาษา
- เริ่มเสียงหลัง user gesture แรกเสมอ (ข้อบังคับ iOS)
- เอฟเฟกต์: สังเคราะห์ด้วย Web Audio (oscillator) — ป๊อบ, ติ๊ง, เสียงเชียร์, ฟองอากาศ — ไม่ต้องมีไฟล์เสียง
- Fallback: ถ้าไม่มี voice ไทยในเครื่อง → ใช้เอฟเฟกต์ + แสดงป้ายชื่อตัวใหญ่แทน

## การทดสอบ

- ตรรกะเกมล้วน (สุ่มโจทย์นับเลข, เลือกตัวลวง, ปรับความยาก) แยกเป็นฟังก์ชัน pure — ทดสอบด้วย `node --test`
- ภาพและการเล่นจริง: เปิดผ่าน Playwright ตรวจทุกเกมก่อนส่งมอบ (screenshot + คลิกจริง)
- ทดสอบบน iPad จริงโดยผู้ใช้หลัง deploy
