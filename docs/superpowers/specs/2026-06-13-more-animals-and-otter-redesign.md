# เพิ่มสัตว์ 4 ตัว + แก้ดีไซน์นากทะเล — Design Spec

วันที่: 2026-06-13
สถานะ: อนุมัติแนวคิดแล้ว รอเขียนแผนพัฒนา
แตะไฟล์หลัก: `js/core/animals.js`

## เป้าหมาย

1. เพิ่มสัตว์อลาสก้า/แปซิฟิก 4 ตัวเข้าทะเบียน `ANIMALS` (จาก 8 เป็น 12) — กระจายเข้าเกมที่สุ่มจากสัตว์โดยอัตโนมัติ
2. วาด SVG ของ **นากทะเล (otter)** ใหม่ให้อ่านออกชัดว่าเป็นนาก (ปัจจุบันเป็นก้อนน้ำตาลลอยหงายท้อง สับสนกับสิงโตทะเล)

ทุกชิ้นเป็น SVG วาดในโค้ด สไตล์ "D" (สัดส่วนอิงสัตว์จริง + สีเรียบ เงา 1-2 ชั้น + ตาโตอ่านอารมณ์ง่าย + กะพริบตา) ตามกฎเดิม ไม่มีไฟล์ภาพ

## ส่วนที่ 1: สัตว์ใหม่ 4 ตัว

เลือกให้รูปทรง/เงาต่างจาก 8 ตัวเดิมและจากกันเองชัดเจน:

| id | TH | EN | ลักษณะภาพ (สไตล์ D) | cry() |
|---|---|---|---|---|
| `octopus` | ปลาหมึกยักษ์ | Octopus | หัว/แมนเทิลกลม สีแดงอมม่วง หนวด 8 เส้นม้วนพลิ้ว มีปุ่มดูดเป็นจุด ตาโตคู่ | `[['bubble'],['pop']]` |
| `jelly` | แมงกะพรุน | Jellyfish | ระฆังโปร่งแสง (ฟ้า/ชมพูอ่อน opacity ต่ำ) ขอบหยัก หนวดเส้นยาวพลิ้วด้านล่าง ตาเล็กน่ารัก | `[['bubble'],['ding']]` |
| `seastar` | ดาวทะเล | Sea Star | ดาว 5 แฉกอวบ สีส้มโอเชอร์ ผิวมีจุดนูน ตาโตคู่ (ทำให้มีชีวิตชีวาแบบ D) | `[['pop'],['ding']]` |
| `shark` | ฉลาม | Shark | ลำตัวเทากระสวย จมูกแหลม ครีบหลังสามเหลี่ยมสูงเด่น หางเสี้ยว เหงือก ตาโต**ยิ้มเป็นมิตร ไม่มีฟันแหลมน่ากลัว** (กฎไม่ทำให้กลัว) | `[['boing'],['bubble']]` |

- ฉลามสีเทา จมูกแหลม ครีบหลังสูง — ต่างจากวาฬเพชฌฆาต (ดำ-ขาว ทรงป้อม) และปลาแซลมอน (เล็ก ชมพู) ชัดเจน
- ดาวทะเลและแมงกะพรุนเติม "เอกลักษณ์รูปทรง" ที่ทะเบียนเดิมไม่มี (ดาว, ระฆังโปร่งแสง)
- แต่ละ object รูปแบบเดียวกับสัตว์เดิม: `{ id, th, en, make(w), cry() }` — `make` ใช้ `svgRoot()` + `el()` + `eye()` helper ที่มีอยู่

## ส่วนที่ 2: แก้ดีไซน์นากทะเล (otter)

วาด `OTTER.make()` ใหม่: **ท่านั่งลอยตัวตั้งขึ้น** (ท่าคลาสสิกของนากทะเลที่ลอยหงายแล้วยกตัวส่วนบน) **ถือเปลือกหอย/หอยไว้บนพุง** หัวกลมใหญ่ ตาโตเด่น จมูกดำ หนวดชัด สีน้ำตาลเดิม (`#6b4a32` family) — ให้ดูออกทันทีว่าเป็นนาก

- viewBox อาจเปลี่ยนเป็นแนวตั้งขึ้น (เช่น ~180×210) ให้เหมาะกับท่านั่ง — `make(w)` รองรับทุก viewBox อยู่แล้ว
- **ต้องตรวจผลกระทบ:** นากเป็น mascot ของการ์ด "วาดตามรอย" บน hub (`game-art.js` วาง otter ที่ตำแหน่ง/ขนาดหนึ่ง) และโผล่ในเกมที่สุ่มสัตว์ — เปลี่ยนสัดส่วนแล้วต้องเช็คว่าวางในการ์ด hub + ในเกมยังสวย ถ้าเพี้ยนให้ปรับตำแหน่งใน `game-art.js` (`'trace-letters'` ใช้ otter)

## การกระจายอัตโนมัติ (ไม่ต้องแตะโค้ดเกม)

เพิ่มลง array `ANIMALS` → เกมเหล่านี้ได้สัตว์ใหม่ทันที (สุ่มจากความยาว array):
- **แตะทะเล** (`tap-sea`) — สุ่มสัตว์โผล่
- **จับคู่เงา** (`shadow-match`) — `pickRound(ANIMALS.map(id), 3)`
- **นับสัตว์ทะเล** (`count-tap`) — สุ่มจาก ANIMALS
- **เสียงเรียกใคร** (`listen-find`) — `CHOICES=4` จาก ANIMALS

ไม่เปลี่ยน: **สะกดคำ** (ใช้ `spell-words.js`) และ **วาดตามรอย** (ใช้ letters data) — เพิ่มคำ/รูปสำหรับสัตว์ใหม่เป็นงานภายหลัง (นอกขอบเขต)

## สถาปัตยกรรม + ไฟล์

| ไฟล์ | บทบาท |
|---|---|
| Modify `js/core/animals.js` | เพิ่ม OCTOPUS/JELLY/SEASTAR/SHARK ใน `ANIMALS`; เขียน `OTTER.make()` ใหม่ |
| Create `tests/animals.test.mjs` | guard: 12 ตัว, id ไม่ซ้ำ, ทุกตัวมี th/en + `make`/`cry` เป็น function |
| (อาจ) Modify `js/core/game-art.js` | ปรับตำแหน่ง otter ในการ์ด trace-letters ถ้าสัดส่วนใหม่ทำให้เพี้ยน |

## การทดสอบ

- `tests/animals.test.mjs` (pure, ไม่เรียก `make()` เพราะ Node ไม่มี DOM — ตรวจเฉพาะรูปร่างข้อมูล): `ANIMALS.length === 12`, id ไม่ซ้ำ, ทุกตัว `typeof make === 'function' && typeof cry === 'function'`, มี th/en, `cry()` คืน array
- เทสต์เดิม 33 ตัวต้องผ่าน (ไม่มีตัวไหน hardcode จำนวน 8 — ยืนยันตอนทำ)
- **Playwright (จริง):**
  - เปิด `?preview=animals` → ดูทั้ง 12 ตัว: 4 ตัวใหม่อ่านออก/สไตล์กลมกลืน + นากตัวใหม่ดูเป็นนากชัด (screenshot)
  - hub: การ์ด "วาดตามรอย" (mascot นาก) ยังสวย
  - เล่นเกมที่สุ่มสัตว์ (เช่น จับคู่เงา/นับ/เสียงเรียกใคร) ดูว่าสัตว์ใหม่โผล่ได้จริง, console 0 error
  - จอ 1180×820 + 844×390

## นอกขอบเขต (บันทึกไว้)

- เพิ่มคำสะกด (เช่น SHARK, STAR) ใน `spell-words.js` และคำ/รูปในเกมวาดตามรอย
- ปรับสมดุลความยาก/จำนวนตัวเลือกในเกมต่าง ๆ หลังสัตว์เพิ่ม (ผู้ใช้เคยบอกว่าจะทบทวนทุกเกมหลังเพิ่มสัตว์ — ทำเป็นรอบแยก)
- รูป pseudo-3D / โลกธีมใหม่
